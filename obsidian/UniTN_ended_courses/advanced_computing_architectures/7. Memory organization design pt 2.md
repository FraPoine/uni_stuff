## Improving cache performance
consider
- Miss rate
- Miss penalty
- Hit time

### Increase cache size
- Decreases miss rate
- May increase hit time
	- large cache might increase hit time just because is large
### Increase block size
- Larger block size reduces compulsory misses (because of spatial locality)
- Larger block size increases miss penalty (larger block to read from memory)
- Larger block size implies fewer blocks for a fixed cache size, which may increase conflict misses and capacity misses (if cache is small)

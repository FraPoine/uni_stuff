## Facial processing
*Facial processing is the discipline that takes facial images / videos / data as input of a system and returns higher level information*
### 2D Facial Expression Recognition (FER)
*FER is the technlogy that analyzes facial expressions from both static images and videos in order to reveal information on one’s emotional state (EDS TechDispatch, issue 1, 2021)*

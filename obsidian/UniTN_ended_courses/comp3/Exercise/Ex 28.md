Exercise 28 Consider the following language: 
- S = {(x ∈ {0, 1}∗, k ∈ N) : x contains a subsequence of k adjacent 0’s }
- For example, (101000101, 3) ∈ S because the binary string contains 3 consecutive  zeroes, while (101000101, 4)̸ ∈ S because the binary string does not contain 4 consecutive zeroes.
1) Prove S ∈ P
2) Prove S ∈ L
*Hint — Again, both points can be proved by describing an algorithm and showing that it has the required property.*

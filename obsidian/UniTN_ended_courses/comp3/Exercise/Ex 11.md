The following statement is actually true, but what’s wrong with the proof provided here? Theorem: **NPSPACE ⊆ PSPACE** 

Proof — Let L ∈ NPSPACE, let N be the NDTM that decides it in polynomial space, and let x be an input string. Since we have no time bounds, we can emulate all possible computations of N (x), one after the other, until one of them accepts x or we exhaust all of them. Of course, there is an exponential number of computations, but we have no time bounds, and each computation only requires polynomial space by definition: the tape can be reused between computations. 
Therefore, we can emulate N by a deterministic, polynomial-space TM M, and L ∈ PSPACE, thus proving the assertion

### Difference between NDTM and DTM
#### DTM
A **DTM** is a theoretical computing model where **each configuration (state + tape symbol)** leads to **exactly one next move**.

#### NDTM
An **NDTM** is a variation where the machine can choose between **multiple possible moves** from the same configuration.

## Savitch’s Theorem
Given a function f (n), NSPACE(f (n)) ⊆ DSPACE(f (n)^2).
***Proof***
- lets consider 
	- a Language L ∈ NSPACE(f(n))
	- generic input x
- there is a NDTM N that decides x ∈ L by using at most O(f|x|) tapes cells. 
- the number of different configurations of the machine is therfore bounded by Nc = 2^O{f|x|}.
- Let us consider the directed graph G = (V, E) having all possible Nc configurations as the set V of nodes, and with an edge (c1, c2) ∈ E if there is a transition rule in the N DT M that allows transition from c1 to c2.
	- Every path in G represents a possible computation of the machine, starting from an arbitrary configuration
- Let us call s ∈ V the initial configuration of N with input x. 
	- Obviously, x ∈ L if and only if there is an accepting computation of N with input x
-  Let us add a new node t to V , and an edge from every accepting state to t.
- At this point, x ∈ L if and only if there is a path from s to t in G, therefore if and only if (G, s, t) ∈ STCON.

Let Σ be a finite alphabet, and 
- LR1, 
- LR2, 
- LRE1, 
- LRE2 
	- ⊆ Σ* 
be four languages on Σ. 
LR1 and LR2 are recursive, 
while LRE1 and LRE2 are recursively enumerable, but not recursive.


For each of the following languages, state if they are recursive, recursively enumerable, or none, and motivate your answers: 
- L1 = LR1 ∪ LR2; 
- L2 = LR1 ∩ LR2; 
- L3 = LR1 ∪ LRE1; 
- L4 = LR1 ∩ LRE1;
- L5 = LRE1 ∪ LRE2;
- L6 = LRE1 ∩ LRE2.

State whether the following properties of Turing machines are computable or not, and motivate your statements
- P1 = {M : M decides LR1};
- P2 = {M : M decides LRE1};
- P3 = {M : if |x| < 100, then M decides x ∈ LR1 in no more than |x|2 + 1 steps};
- P4 = {M : if |x| < 100, then M decides x ∈ LRE1 in no more than |x|2 + 1 steps}

----
Lets start with the definition of language, recursive and recursively enumerable

**Language**
**language** is simply a **set of strings** over some finite alphabet.

**Recursive**
A language L over an alphabet Σ is **recursive** if exists a TM M such that for every string w ∈ Σ*
- M halts in a finite number of steps
- Accepts w if w ∈ L
- Reject w if w ∉ L

**Recursively enumerable**
A language is **recursively enumerable** if there exists a TM that will
- Accept any string that belongs to the language
- may not halt on string not in the language

	- this one is still RE because it accepts the input only if is accepted by both the machine that decide the two language, in every other case or loop endlessly or halt after a rejection, and for the exact same reason the other two language L5 and L6 are RE

## 2
computable or not computable
rices theorem
- all non-trivial semantic properties of TMs are undecidable

**semantic**
- a property is semantic if its value is shared by all TMs recognizing the same language


- P1 = {M : M decides LR1};
	- Semantic
	- not trivial
	- undecidable
- P2 = {M : M decides LRE1};
	- Semantic
	- its trivial because we state that LRE1 is not recursive so does not exists any machine that can decide LRE1, and so P2 is empty, so P2 is trivially recursive
- P3 = {M : if |x| < 100, then M decides x ∈ LR1 in no more than |x|2 + 1 steps};
	- not semantic
	- what does it mean P3
		- if the number of the input is lower than 100, then M decides x ∈ LR1 in no more than input size \* 2  + 1 steps
	- we have that LR1 is recursive
	- to verify P3 we need to iterate every possible string with size less than 100 and then check the response
	- so is computable 
- P4 = {M : if |x| < 100, then M decides x ∈ LRE1 in no more than |x|2 + 1 steps}
	- not semantic
	- Because LRE1​ is **recursively enumerable but not recursive**, we can’t in general determine whether a given string belongs to it. Checking correctness for those small inputs would require solving the **membership problem** for LRE1​, which is **undecidable**.

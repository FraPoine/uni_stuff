Consider the following language on the two-symbol alphabet {0, 1}:
- L = {0^n \* 1^m | n, m ∈ N ∧ n > m}. 
In plain terms, a string is in L if and only if it starts with a sequence of 0’s followed by a (possibly empty) sequence of 1’s and nothing else, with strictly more 0’s than 1’s.
Some examples: 
- 00011 ∈ L 
- 00111 ∉ L
- 0 ∈ L
- 1 ∉ L 
- 10 ∉ L 
- 11000 ∉ L 
- 0110100 ∉ L 
- 0000 ∈ L 
- 0011 ∉ L
- ε ∉ L

1) Write down a one-tape deterministic Turing Machine M on the three-symbol alphabet {0, 1, -} that, given an input string s ∈ {0, 1}∗, decides s ∈ L. 
	- You may assume that the input string s is surrounded by infinite 
2) What is the time complexity of your machine M? 
	- More precisely: if n is the input size, what is the smallest exponent k such that M ∈ DTIME(n^k)? Explain briefly.

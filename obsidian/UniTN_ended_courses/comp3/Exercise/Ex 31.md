Let L be a finite, non-empty language on the alphabet Σ = {0, 1}. 
For each of the following propositions say if it is true, false or (to the best of our knowledge) unknown, and briefly motivate your answer. 
1. L ∈ P
2. L ∈ L
3. L is polynomial-time reducible to 3SAT
4. 3SAT is polynomial-time reducible to L

### 1 
L ∈ P
On L we know
- its finite
- its not empty
- its on the alphabet Σ = {0,1}

can i find a solution in polynomial time?
- Yes — **every finite language (over any alphabet) is in P**
- A language L is in P if there is a deterministic Turing machine (or algorithm) that, given an input string w of length n, decides whether w∈L in time polynomial in n.

### 2
L ∈ L

The answer is **yes** — all finite languages are in L too.
You can store a constant-sized lookup table (because L is fixed and finite) in the _finite control_ of a Turing machine, and then just scan the input to check for equality against each word.

Space usage: You only need O(log⁡ n) (in fact O(1)) extra work tape space to keep track of your position and a few comparison flags.

Time: still O(n), which is fine for L.

### 3 
L is polynomial-time reducible to 3SAT.

Yes — you can absolutely say that.

In fact, it’s almost _too_ easy for finite languages.
A **polynomial-time many-one reduction** from L to 3SAT means we need a function
- f: {0,1}* --> 3SAT instances 
computable in polynomial time such that x ∈ L <==> f(x) ∈ 3SAT

since L is finite and fixed 
- if x ∈ L, map it to some fixed 3CNF formula Fyes
- if x ∉ L, map it to some fixed 3CNF formula Fno
- checking whether x is in L or not require O(n), so computing f(x) is still polynomial 
Thus, L ≤p​ 3SAT.

### 4
3SAT is polynomial-time reducible to L

If 3SAT is polynomial-time reducible to L, it means that L itself is NP-complete. Since we know that L ∈ P, this is true if and only if P = NP. Hence the requested reduction is, to the best of our knowledge, unlikely.

A graph G = (V, E) is connected if there is a path between every pair of nodes. Show that the language of connected graphs is in NL.

---
we have 
- graph G = (V, E)
- CONNECTED:
	- if there is a path between any pair of nodes 
- its a problem of ST-CONNECTIVITY

- we can use the BFS – Breadth-First Search
	- in order to map all the connection of G and became easy to verify if exists a path between every pair of nodes
- we already know to STCON to be NL, we only need to iterate pairs of nodes, so the complexity became c (constant) log(n)^2

### **STCON è risolvibile in tempo polinomiale**

Come abbiamo detto prima, puoi usare **BFS o DFS** per capire se c’è un cammino da sss a ttt. Questi algoritmi hanno tempo:
- O(∣V∣+∣E∣)
Quindi STCON è in P, cioè risolvibile in tempo polinomiale.

### **Perché STCON è in NL?**

#### Intuizione:
- Immagina una macchina che deve decidere se esiste un cammino da sss a ttt.
- La macchina può **indovinare** (non deterministicamente) il prossimo nodo nel cammino, passo dopo passo.
- A ogni passo, la macchina salva solo:
    - il nodo corrente,
    - un contatore del numero di passi fatti (per non ciclare all’infinito),
- Entrambi possono essere memorizzati usando **solo O(log∣V∣) spazio**, perché devi memorizzare solo indici di nodi (numeri da 1 a |V|) e un contatore massimo.

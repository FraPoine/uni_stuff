### Watermark detection
- Compute the DCT of the watermarked (and possibly attacked) cover image I*
- Extract the watermark X*
	- Additive - SS : vi ' = vi + αxi 
	- Multiplicative - SS : vi ' = vi(1+ αxi ) i =1,...,n 
- Evaluate the similarity of X* and X using
	- sim(X, X*) = $X*X\fract(\rad(X*X)$
- If sim (X,X*) > T, a given threshold, the watermark exists
- Setting the detection threshold is a classical decision estimation problem in which we wish to minimize both the rate of false negatives (missed detection) and false positives (false alarms).
- Other possible measures are standard correlation coefficient or normalized correlation.

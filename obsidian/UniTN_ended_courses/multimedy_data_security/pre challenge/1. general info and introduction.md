### Lecture 2
Application
- Watermarking
	- the practice of imperceptibly altering a work to embed a message about that work 
	- from the application view point its not born to be something like a metadata, its in the file :)
	- A digital watermark is an identification code bearing information (logo’s, signatures) about the copyright owner, authorized consumers and so on.
- Fingerprinting
	-  insert different marks into different copies of the same document
	- the difference with watermarking is that I want to create different copies of the same image, its like an identifier, like to recognize the one who send the image.
- Steganography
	- (covered writing) the process of secretly embedding information into a data source in such a way its very existence is concealed
	- we will learn how to insert a logo in an image
	- we care more about the message and less about the quality of the image
- Cryptography
	- rendering messages unintelligible to any unauthorized persons who might intercept them
- Digital forensic
	-  automatic analysis of data source and possible tampering
